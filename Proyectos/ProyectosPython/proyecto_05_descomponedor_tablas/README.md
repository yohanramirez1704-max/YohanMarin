# 🧮 Descomponedor Numérico y Generador de Tablas Avanzado

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Loops](https://img.shields.io/badge/Bucles-Anidados-teal?style=for-the-badge)

Estructura modular orientada al procesamiento aritmético y manipulación de cadenas que descompone números de 6 dígitos en subpares independientes, ejecutando secuencias matriciales de multiplicación.

## ✨ Características Destacadas
* **Manipulación de Strings Avanzada:** Segmentación posicional precisa en variables independientes (A, B y C) mediante indexación directa.
* **Algoritmo de Bucles Anidados:** Motor matemático que genera secuencias completas de multiplicación cruzada (del 1 al 10) abarcando dinámicamente todo el rango numérico delimitado entre el subpar A y B.
* **Función de Reconstrucción Lógica:** Combina segmentos numéricos discontinuos para construir identificadores totalmente nuevos por medio de operaciones matemáticas directas.
