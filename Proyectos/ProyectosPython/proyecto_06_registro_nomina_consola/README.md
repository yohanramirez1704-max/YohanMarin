# 💼 Sistema de Procesamiento de Nómina Empresarial (Consola)

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Data](https://img.shields.io/badge/Estructura-Diccionarios--Lista-orange?style=for-the-badge)

Una herramienta robusta de terminal enfocada a la gestión interna de recursos humanos. Permite el registro secuencial de empleados y automatiza la auditoría de salarios y métricas de desempeño corporativo.

## ✨ Características Destacadas
* **Estructura de Datos Híbrida:** Almacena los perfiles individuales de los trabajadores mediante diccionarios asociativos, compilados de forma global dentro de listas dinámicas.
* **Validaciones con Expresiones Regulares (Regex):** Sanitización estricta de entradas de texto mediante patrones de coincidencia para rechazar caracteres especiales o números en nombres y departamentos.
* **Motor Estadístico Comercial:** Realiza análisis lineales sobre el set de datos para deducir de forma inmediata el coste bruto de nómina y el perfil con mayor remuneración económica.
