# 🏢 Payroll Management System (Advanced POO)

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![POO](https://img.shields.io/badge/POO-Herencia_M%C3%BAltiple-purple?style=for-the-badge)
![GUI](https://img.shields.io/badge/UI-Tkinter_TTK-lightgrey?style=for-the-badge)

Solución empresarial avanzada con interfaz gráfica para la gestión y liquidación de salarios basada en polimorfismo y herencia múltiple.

## ✨ Características Destacadas
* **Jerarquía de Clases Completa:** Diseñado bajo una clase abstracta padre (`Empleado`) que ramifica de manera polimórfica en perfiles de tiempo completo, contrataciones por horas y esquemas comerciales de comisión.
* **Herencia Múltiple e Interfaces:** Implementación de la clase funcional Mixin `Bonificable` para dotar a clases específicas de la capacidad de acumular incentivos financieros cruzados.
* **UI Empresarial Modificada:** Listado de datos enriquecido utilizando componentes visuales nativos, cuadros de entrada numéricos protegidos y selectores de visualización de detalles en pantalla.
