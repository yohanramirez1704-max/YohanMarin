# 📊 Analizador Matemático de Matrices Bidimensionales

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Math](https://img.shields.io/badge/Estructura-Matrices-green?style=for-the-badge)

Aplicación matemática en consola encargada del llenado, validación y procesamiento estadístico avanzado de una matriz de dimensiones fijas de $4 \times 4$.

## ✨ Características Destacadas
* **Estructuración Bidimensional:** Generación de listas anidadas interactivas en tiempo de ejecución.
* **Mapeo con Diccionarios de Conteo:** Implementación eficiente de colecciones clave-valor para rastrear la frecuencia absoluta de apariciones numéricas.
* **Potenciación por Frecuencia:** Algoritmo dinámico que calcula la potencia exponencial tomando la base numérica original elevada a su propio índice de repetición.

## ⚙️ Flujo del Programa
1. **Captura Limitada:** Solicita datos posición por posición validando rangos estrictos entre 3 y 6 mediante control de excepciones (`ValueError`).
2. **Conteo Exhaustivo:** Itera sobre la matriz indexada mapeando los resultados en diccionarios.
3. **Cálculo de Potencias:** Muestra los resultados matemáticos finales formateados.
